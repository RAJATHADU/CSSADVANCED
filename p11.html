CSS Pseudo-Elements

The CSS pseudo elements allow you to style the elements or parts of the elements without adding any IDs or classes to them.

CSS3 introduced a new doubled-colon(::) syntax for pseudo-elements to distinguish between them and pseudo-classes.

selector::pseudo-element {property:value;}
